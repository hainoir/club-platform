# Frontend Developer Club Management Platform

This is a polished UI prototype developed using Next.js 15, Tailwind CSS, Lucide React, and shadcn/ui.

## 🚀 启动指南 (Getting Started)

由于在环境生成阶段跳过了自动安装步骤，项目代码已经全部就绪。请按以下命令手动启动：

1. **安装依赖 (Install Dependencies)**
   ```bash
   npm install
   # 或者使用 pnpm install / yarn
   ```

2. **启动开发服务器 (Start Development Server)**
   ```bash
   npm run dev
   # 或者使用 pnpm dev / yarn dev
   ```

3. **查看效果**
   打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## ✨ 特性 (Features)
- 🌓 完全支持亮色与暗色模式的优雅切换
- 📱 响应式设计，完美适配手机、平板与桌面显示
- ⚡ 零延迟错觉的顺滑客户端状态模拟（基于 Toast 和虚拟进度反馈）
- 🧩 包含 Dashboard、数据表格(Member Management)、网格卡片流(Events) 等典型管理端交互设计。
